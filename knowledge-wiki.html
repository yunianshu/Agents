人工智能基础

人工神经网络 核心

由输入层、隐藏层、输出层组成

  • 输入层到输出层是< span class="highlight">单向流动
  • 1986年 BP算法（反向传播）
  • 通过调整权重使输出接近目标值
深度学习 重点

深度学习 = 机器学习 + 特征提取

  • 多隐藏层的神经网络
  • 自动学习基础特征和复杂特征
  • 比传统机器学习多了特征工程部分
机器学习分类 定义
  • 监督学习 - 有标签数据
  • 无监督学习 - 无标签（聚类）
  • 半监督学习 - 少量标签
  • 深度学习 - 多层神经网络
  • 强化学习 - 奖励驱动
  • 集成学习 - 多模型组合
集成学习方法 重要

Bootstrap + 弱分类器组合

  • AdaBoost - 权重根据错误率调整
  • Stacking - 元分类器预测结果

深度学习网络架构

卷积神经网络 CNN 核心 📷 图像处理

主要用于图像和视频数据

  • 卷积层 - 提取特征
  • 池化层 - 下采样，减少尺寸
  • 全连接层 - 输出分类结果

应用：图像分类、目标检测、语义分割、视频处理

循环神经网络 RNN 核心 📝 序列数据

处理序列数据（文本、语音）

  • 捕获输入序列的上下文信息
  • 是对方向传播算法的优化
长短时记忆网络 LSTM 重点

解决 RNN 的梯度消失/爆炸问题

  • 输入门 - 控制信息进入
  • 遗忘门 - 控制信息遗忘
  • 输出门 - 控制信息输出

应用：语音识别、文本生成、视频描述

Transformer 核心 🔥 现代主流

基于自注意力机制的深度学习模型

  • 编码器 + 解码器结构
  • 多层自注意力 + 前馈神经网络

应用：NLP、计算机视觉、语音处理

CNN 经典架构 必须掌握
📜 LeNet - AlexNet - VGG ▼

LeNet - 最早发布的CNN，手写数字识别

AlexNet - 第一个用于图像分类的CNN (2012)

VGG - 3×3小卷积核 + 2×2最大池化，堆叠增加深度

📜 GoogleNet - ResNet - DenseNet ▼

GoogleNet/Inception - 提高计算效能，减少参数

ResNet - 残差网络，解决梯度消失，可训练极深网络

DenseNet - 密集连接，特征再利用，减少参数

CNN 结构特性 定义
  • 局部连接 - 只连接局部区域
  • 权重共享 - 减少参数数量
  • 空间/时间次采样 - 降维

卷积层是3维结构，可显著减少连接数

点云数据处理 重要

三维可视化 + 深度学习输入

  • 采集设备：激光扫描仪、深度相机、双目相机
  • 应用：三维重建、自动驾驶、地物分割

关键点算法：Harris3D、SIFT3D、NARF(最常用)

滤波方法：双边滤波、高斯滤波、条件滤波、直通滤波

机器学习模型评估

分类模型评估指标 核心

准确率 = 预测正确 / 总样本

accuracy = accuracy_score(y_true, y_pred)

⚠️ 不适用于类别不平衡场景

精确率(查准率) = TP / (TP+FP)

预测为正类中实际为正类的比例

precision = precision_score(y_true, y_pred)

场景：推荐系统、搜索引擎

召回率(查全率) = TP / (TP+FN)

实际正类中被正确预测的比例

recall = recall_score(y_true, y_pred)

场景：罕见病检测、欺诈检测

F1分数 = 2×精确率×召回率 / (精确率+召回率)

调和平均值，取值 [0, 1]

f1 = f1_score(y_true, y_pred, average='binary')
ROC 曲线与 AUC 重点

ROC曲线 - 展示不同阈值下的分类能力

  • 纵轴：TPR（真正类比例）
  • 横轴：FPR（假正类比例）

AUC = ROC曲线下面积

  • AUC = 1 → 完美分类
  • AUC = 0.5 → 随机猜测
fpr, tpr, thresholds = roc_curve(y_true, y_scores) roc_auc = auc(fpr, tpr)
回归模型评估 核心
指标公式特点
MAE平均绝对误差对异常值不敏感
MAD中值绝对误差适合大量异常值
MSE均方误差对离群值敏感
R²决定系数R²=1完美，R²<0模型有问题
mae = mean_absolute_error(y_true, y_pred) mse = mean_squared_error(y_true, y_pred) r2 = r2_score(y_true, y_pred)
聚类模型评估 重点

轮廓系数

  • 接近 1 → 聚类效果好
  • 接近 0 → 在簇边界
  • 接近 -1 → 可能分配错误

ARI (调整兰德指数)

  • 取值 [-1, 1]
  • ARI = 1 → 完全一致
  • ARI ≈ 0 → 随机水平
silhouette = silhouette_score(X, labels) ari = adjusted_rand_score(true_labels, pred_labels)
sklearn 核心 API 公式
  • fit() - 训练模型
  • get_params() - 获取参数
  • set_params() - 设置参数
  • score() - 评估模型
get_params() 返回字典，对超参数调优很有用
数据预处理 定义

标准化 - 缩放到均值0，方差1

归一化 - 缩放到单位范数（长度1）

数据处理流程

数据处理基本流程 核心
采集 → 清洗 → 标注 → 审核 → 质检 → 验收
文本类数据 重要

预处理：清洗、分词、句法分析

特征工程：词频、词向量

特点：

  • 数据量大、高维稀疏
  • 非结构化、语义复杂
  • 应用：舆情分析、智能客服
视觉类数据 重要

处理方法：

  • 图像：增强、校正、裁剪、色彩调整
  • 视频：分段、关键帧提取、压缩

客观审核指标：

  • PSNR - 峰值信噪比（值越大越好）
  • SSIM - 结构相似度
  • VMAF - 视频质量多方法评价融合
语音类数据 重要

预处理方法：滤波、降噪、平滑、加重

语音段落切割规范：

  • 以说话人整句话为单位切割
  • 在波形图最低点设置边界
  • 保留 2-3 秒静音段
数据标注类型 定义

文本标注：序列标注、关系标注、属性标注、类别标注

视觉标注：

  • 关键点标注 - 人脸、姿态
  • 矩形框标注 - 目标检测
  • 区域标注 - 语义分割
  • 属性标注 - 标签分类

标注质量要求：像素误差 < 1个像素

数据标注通用规则 必须记住
  • 贴边 - 标注框紧贴对象边缘
  • 独立 - 每个对象单独标注框
  • 不越界 - 标注框不能在图片边界
  • 确定共同特征 - 观察找规律
数据集格式 格式
  • VOC - XML格式，易解析
  • COCO - JSON格式，支持多任务
  • TFRecord - 二进制，大规模数据

标注工具：CVAT、Labelme、LabelStudio

数据清洗质量评估 重要
  • 完整性 - 无缺失
  • 准确性 - 无错误异常
  • 一致性 - 格式统一
  • 有效性 - 取值在范围内
  • 唯一性 - 无重复记录
