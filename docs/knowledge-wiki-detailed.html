第一章：人工智能基础

人工神经网络、机器学习分类、集成学习、注意力机制

人工神经网络 ANN 核心概念

定义：模拟人脑神经系统工作原理的数学模型，由大量神经元互联形成。

三层结构：

  • 输入层 - 接收外部数据，不进行计算
  • 隐藏层 - 核心计算层，可有多层
  • 输出层 - 输出最终结果

数据流向：

输入层 → 隐藏层₁ → 隐藏层₂ → ... → 输出层
⚠️ 重要

输入层到输出层是单向流动，信号向前传播

BP 反向传播算法 核心算法 1986年

全称：Backpropagation Algorithm（反向传播算法）

核心原理：

  • 通过损失函数计算输出与真实值的误差
  • 误差从输出层向输入层反向传播
  • 根据梯度下降法调整各层权重
权重更新公式
W_new = W_old - η × ∂L/∂W

其中 η 为学习率，L 为损失函数

训练过程：

前向传播 → 计算误差 → 反向传播 → 更新权重
深度学习 Deep Learning 核心概念

定义：深度学习是机器学习的一个分支，是传统人工神经网络发展到多隐藏层的情况。

深度学习 vs 传统机器学习：

对比项机器学习深度学习
特征提取人工设计特征自动学习特征
特征复杂度简单基础特征基础+复杂特征
网络结构浅层网络深层网络
计算资源要求较低需要GPU
💡 理解

深度学习 = 机器学习 + 自动特征提取

核心优势：自动从数据中学习层次化特征表示

机器学习分类 核心框架

六大分类：

1. 监督学习 Supervised Learning

有标签数据，学习输入到输出的映射关系

应用：分类、回归

2. 无监督学习 Unsupervised Learning

无标签数据，发现数据内部结构

应用：聚类、降维

3. 半监督学习 Semi-supervised Learning

少量标签 + 大量无标签数据

应用：图像分类、文本处理

4. 深度学习 Deep Learning

多层神经网络，自动学习特征

应用：CV、NLP、语音

5. 强化学习 Reinforcement Learning

通过奖励信号学习策略

应用：游戏、机器人

6. 集成学习 Ensemble Learning

组合多个学习器提升性能

方法：Bagging、Boosting、Stacking

集成学习方法 重要方法

核心思想：将多个弱学习器组合，形成更强大的学习器

1. Bootstrap + Boosting（AdaBoost）

  • 迭代训练多个弱分类器
  • 每个分类器的权重根据错误率调整
  • 错误率高的分类器权重降低
  • 最终组合所有分类器输出

2. Stacking（堆叠集成）

  • 将多个学习器的输出作为新特征
  • 使用元分类器进行最终预测
  • 典型结构：基学习器 → 元学习器
模型₁ 模型₂ 模型₃ → 元特征 → 元分类器 → 输出
注意力机制 vs 自注意力机制 核心差异

已知：query（查询）和序列

目标：从序列中找出和查询最相关的信息

特点：

  • 应用在 output 层或 state 层
  • 连接两个不同组件（如 Encoder → Decoder）
  • 一个模型中通常只使用 1 次
  • 善于寻找两个序列之间的关系
  • 可以连接两种不同模态（图片+文字）

特点：

  • 更多关注 input 层
  • 只关注一个组件（如 Transformer 的 Encoder）
  • 一个模型中通常使用多次
       例：Transformer 18次，BERT 12次
  • 善于寻找同一序列中不同部分的关系
  • 更多用于同一种模态
维度注意力机制 AT自注意力 SA
关注层级output/state 层input 层
组件数量两个组件单个组件
使用次数通常 1 次通常多次
擅长任务两序列关系同序列关系
模态类型跨模态同模态为主

第二章：深度学习网络架构

CNN、RNN、LSTM、Transformer、经典架构、点云处理

卷积神经网络 CNN 核心网络 📷 图像处理

定义：一种前馈神经网络，受生物学上感受野机制启发提出

三大结构特性：

  • 局部连接 - 只连接局部区域神经元
  • 权重共享 - 同一特征图共享权重
  • 空间/时间次采样 - 降维增强鲁棒性

网络结构：

1. 输入层

接收输入图像，将其转换为特征图

2. 卷积层

通过卷积核提取图像特征

三维结构：高度 × 宽度 × 通道数

3. 池化层

对特征图进行下采样，减少空间维度

  • 最大池化 - 取区域最大值
  • 平均池化 - 取区域平均值
4. 全连接层

连接所有特征，输出最终分类结果

典型应用：

图像分类 目标检测 语义分割 视频处理
循环神经网络 RNN 核心网络 📝 序列数据

全称：Recurrent Neural Network

核心能力：

  • 处理序列数据（文本、语音、时间序列）
  • 捕获输入序列的上下文信息
  • 对方向传播算法的优化

网络结构特点：

X₀ → 隐藏层 → X₁ → 隐藏层 → ...

每个时间步的隐藏层不仅接收当前输入，还接收上一时间步的隐藏状态

⚠️ RNN 问题
  • 梯度消失 / 梯度爆炸
  • 长期依赖问题
长短时记忆网络 LSTM 重点网络

全称：Long Short-Term Memory

提出目的：解决传统 RNN 的梯度消失/梯度爆炸问题

核心机制 - 三门结构：

🚪 输入门 (Input Gate)

控制哪些新信息进入细胞状态

公式：i = σ(Wᵢ·[hₜ₋₁, xₜ] + bᵢ)

🚪 遗忘门 (Forget Gate)

决定细胞状态中哪些信息被丢弃

公式：f = σ(Wf·[hₜ₋₁, xₜ] + bf)

🚪 输出门 (Output Gate)

决定下一隐藏状态的值

公式：o = σ(Wₒ·[hₜ₋₁, xₜ] + bₒ)

典型应用：

  • 语音识别
  • 文本生成
  • 视频描述
Transformer 模型 核心架构 🔥 现代主流

核心特点：基于自注意力机制的深度学习模型

架构组成：

编码器 + 解码器
📦 编码器 (Encoder)

N 层堆叠，每层包含：

  • 多头自注意力层 (Multi-Head Self-Attention)
  • 前馈神经网络层 (Feed Forward)
  • 残差连接 + 层归一化
📦 解码器 (Decoder)

N 层堆叠，每层包含：

  • 掩码多头自注意力层
  • 编码器-解码器注意力层
  • 前馈神经网络层

应用领域：

NLP 计算机视觉 语音处理
CNN 经典架构演进 必须掌握
📜 LeNet - AlexNet - VGG ▼
LeNet (1998)

最早发布的卷积神经网络

用于手写数字识别（MNIST）

结构：7层网络（2卷积+2池化+3全连接）

AlexNet (2012)

ImageNet 竞赛冠军

第一个用于图像分类任务的CNN

创新：ReLU激活、Dropout、GPU训练

VGG (2014)

显著提升图像分类准确率

核心设计：

  • 全部使用 3×3 小卷积核
  • 使用 2×2 最大池化
  • 堆叠多个相同结构增加深度
📜 GoogleNet - ResNet - DenseNet ▼
GoogleNet / Inception (2014)

提高计算效能和模型性能

创新：Inception 模块（多尺度卷积并行）

优势：显著减少参数数量

ResNet (2015)

残差网络

解决问题：梯度消失，使极深网络可训练

核心：残差连接 (Skip Connection)

公式：y = F(x) + x

DenseNet (2017)

密集连接网络

解决问题：梯度消失 + 特征再利用

核心：每层与所有后续层直接连接

优势：减少参数数量

点云数据处理 重要内容

定义：三维空间中的点集合，包含坐标、颜色等信息

采集设备：

  • 深度相机 - 红外激光投影，适合室内AR/VR
  • 双目相机 - 双视角计算三维坐标
  • 激光扫描仪 - 高精度测量

应用领域：

三维重建 自动驾驶 地物分割 形状分类

关键点提取算法：

类型算法
2D关键点Harris角点、SIFT、SURF
3D关键点Harris3D、ISS3D、NARF(最常用)、SIFT3D

点云滤波方法：

双边滤波、高斯滤波、条件滤波、直通滤波、随机采样一致滤波

特征描述算法：

  • 法线和曲率计算
  • 特征值分析
  • PFH / FPFH（点特征直方图）
  • 三维形状语义、旋转图像

配准算法：

NDT、VPF、4PCs、ICP

第三章：机器学习模型评估

分类/回归/聚类评估指标、sklearn API、数据预处理

混淆矩阵 Confusion Matrix 基础框架

定义：分类模型预测结果的可视化矩阵

四要素：

预测为正类 预测为负类
实际正类 TP (真正类) FN (假负类)
实际负类 FP (假正类) TN (真负类)

详细定义：

TP (True Positive)

真实类别是正类，模型识别结果也是正类

FN (False Negative)

真实类别是正类，但模型识别为负类

FP (False Positive)

真实类别是负类，但模型识别为正类

TN (True Negative)

真实类别是负类，模型识别结果也是负类

准确率 Accuracy 核心指标

定义：分类模型正确分类的样本数与总样本数之比

计算公式
Accuracy = (TP + TN) / (TP + TN + FP + FN)

= 预测正确的样本数 / 总样本数

from sklearn.metrics import accuracy_score y_true = [1, 0, 1, 1, 0, 1] y_pred = [1, 0, 1, 0, 0, 1] accuracy = accuracy_score(y_true, y_pred) print(f"准确率: {accuracy:.2f}") # 输出: 0.83
⚠️ 局限性

准确率不适用于类别不平衡的情况

例如：医疗诊断中，99%的正常人被误判为病人可能仍有99%准确率

精确率 Precision (查准率) 重点指标

定义：模型预测为正类的样本中，实际确实为正类的比例

计算公式
Precision = TP / (TP + FP)

= 预测为正类且正确的 / 所有预测为正类的

from sklearn.metrics import precision_score precision = precision_score(y_true, y_pred) print(f"精确率: {precision:.2f}")

适用场景：

  • 推荐系统 - 不希望推荐错误的内容
  • 文献检索 - 不希望返回无关结果
  • 搜索引擎 - 注重搜索结果准确性
召回率 Recall (查全率) 重点指标

定义：所有实际为正类的样本中，被模型正确预测为正类的比例

计算公式
Recall = TP / (TP + FN)

= 预测为正类且正确的 / 所有实际为正类的

from sklearn.metrics import recall_score recall = recall_score(y_true, y_pred) print(f"召回率: {recall:.2f}")

适用场景：

  • 罕见病检测 - 不能漏掉任何病例
  • 欺诈检测 - 不能放过任何欺诈行为
  • 医疗诊断 - 漏诊后果严重
F1 分数 综合指标

定义：精确率和召回率的调和平均，提供两者之间的平衡度量

计算公式
F1 = 2 × (Precision × Recall) / (Precision + Recall)

本质是 Precision 和 Recall 的调和平均

取值范围与含义：

取值含义
F1 = 1最佳，精确率和召回率都为1
F1 = 0.5中等水平
F1 = 0最差，精确率或召回率至少有一个为0
from sklearn.metrics import f1_score # 二分类 f1 = f1_score(y_true, y_pred, average='binary') # 多分类 f1_macro = f1_score(y_true, y_pred, average='macro') f1_micro = f1_score(y_true, y_pred, average='micro')
💡 何时使用 F1

当需要平衡精确率和召回率时使用，特别是类别不平衡的数据集

平均准确率 Balanced Accuracy 不平衡数据

定义：对每个类别的召回率取平均，解决类别不平衡问题

核心优势：

  • 更关注少数类的表现
  • 提供更公平的评价标准
  • 适合误分类成本不对称的场景
from sklearn.metrics import balanced_accuracy_score balanced_acc = balanced_accuracy_score(y_true, y_pred) print(f"平均准确率: {balanced_acc:.2f}")
ROC 曲线与 AUC 核心指标
📈 ROC 曲线

全称：Receiver Operating Characteristic Curve（受试者工作特征曲线）

纵轴：TPR（真正类比例）= TP/(TP+FN)

横轴：FPR（假正类比例）= FP/(FP+TN)

展示模型在不同阈值下的分类能力

📊 AUC

全称：Area Under the ROC Curve

定义：ROC曲线下的面积

AUC 取值与含义：

AUC值含义
AUC = 1.0完美分类
AUC = 0.9~1.0优秀
AUC = 0.7~0.9良好
AUC = 0.5随机猜测（基准线）
AUC < 0.5比随机猜测差
from sklearn.metrics import roc_curve, auc # roc_curve 返回 fpr, tpr, thresholds fpr, tpr, thresholds = roc_curve(y_true, y_scores) roc_auc = auc(fpr, tpr) print(f"AUC: {roc_auc:.2f}")
回归模型评估指标 核心指标
📊 MAE 平均绝对误差 ▼

公式：MAE = (1/n) × Σ|yᵢ - ŷᵢ|

特点：

  • 对异常值不敏感
  • 衡量预测值与真实值的平均绝对偏差
from sklearn.metrics import mean_absolute_error mae = mean_absolute_error(y_true, y_pred)
📊 MSE 均方误差 ▼

公式：MSE = (1/n) × Σ(yᵢ - ŷᵢ)²

特点：

  • 对离群值比较敏感
  • MSE 越小，模型拟合效果越好
from sklearn.metrics import mean_squared_error mse = mean_squared_error(y_true, y_pred)
📊 R² 决定系数 ▼

公式：R² = 1 - Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²

含义：

R²值含义
R² = 1完美拟合
R² = 0无法解释任何变异
R² < 0模型选择错误或特征不当
from sklearn.metrics import r2_score r2 = r2_score(y_true, y_pred)
📊 MAD 中值绝对误差 ▼

公式：MAD = median(|yᵢ - ŷᵢ|)

适用：数据集中存在大量异常值的情况

from sklearn.metrics import median_absolute_error mad = median_absolute_error(y_true, y_pred)
聚类模型评估 重点内容
🎯 轮廓系数 Silhouette Score

公式：s = (b - a) / max(a, b)

参数：

  • a = 样本与同簇其他样本的平均距离
  • b = 样本与最近簇中样本的平均距离

取值含义：

取值含义
接近 +1聚类效果好，簇内紧密，簇间分离
接近 0样本在两个或多个簇的边界上
接近 -1样本可能被分配到错误的簇中
from sklearn.metrics import silhouette_score silhouette = silhouette_score(X, labels)
🎯 ARI 调整兰德指数

全称：Adjusted Rand Index

取值范围：[-1, 1]

取值含义
ARI = 1两个聚类结果完全一致
ARI ≈ 0相似度不比随机分配更好
ARI < 0相似度低于随机预期
from sklearn.metrics import adjusted_rand_score ari = adjusted_rand_score(true_labels, predicted_labels)
sklearn 核心 API 必须掌握

核心方法：

fit() - 训练模型

接受训练数据，调整模型内部参数以优化目标函数

model.fit(X_train, y_train)
get_params() - 获取参数

返回包含模型所有可配置参数及当前值的字典

params = model.get_params() # 调试、记录模型配置、网格搜索调优
set_params() - 设置参数

用于设置模型参数

model.set_params(**params)
score() - 评估模型

返回模型评估分数

score = model.score(X_test, y_test)
💡 优势

sklearn 提供了一套统一且直观的接口

支持模型训练、参数获取、评估、设置

第四章：数据处理流程

采集→清洗→标注→审核→质检→验收 | 文本/视觉/语音数据处理

数据处理基本流程 核心框架

六步骤流程：

①采集 → ②清洗 → ③标注 → ④审核 → ⑤质检 → ⑥验收
1. 数据采集

从各种来源收集原始数据

2. 数据清洗

处理缺失值、异常值、重复数据

3. 数据标注

为数据添加标签（分类、检测、分割等）

4. 数据审核

验证标注质量、一致性

5. 数据质检

最终质量检查

6. 数据验收

交付合格数据

文本类数据处理 重点内容

数据类型：

  • 数值型数据
  • 日期时间型数据
  • 逻辑型数据
  • 文本型数据（非结构化）

预处理流程：

清洗 分词 去停用词 词性标注 实体识别

特征工程：

  • 词频统计 (TF)
  • 逆文档频率 (IDF)
  • 词向量 (Word2Vec, BERT)

文本数据特点：

特点说明
数据量大以大规模形式存在
高维稀疏性维度高且分布稀疏
非结构化缺乏固定格式
语义复杂性含情感、上下文、隐含信息

应用场景：

舆情分析、智能客服、语义分析、机器翻译

视觉类数据处理 重点内容

数据类型：

  • 图像类 - 人脸、文字、商标、手写体等
  • 视频类 - 包含图像、语音、音乐、音效、文字

处理方法：

🖼️ 图像处理
  • 增强 - 亮度、对比度调整
  • 校正 - 几何校正、畸变校正
  • 裁剪 - 去除无用区域
  • 色彩调整 - 白平衡、饱和度
  • 模糊/锐化 - 图像质量调整
🎬 视频处理
  • 分段 - 按场景/时间切分
  • 关键帧提取 - 提取代表性帧
  • 压缩 - 视频编码压缩

客观审核指标：

指标全称说明
PSNR峰值信噪比值越大图像质量越好
SSIM结构相似度评估压缩/去噪/增强效果
VMAF视频质量评价融合深度学习方法的视频质量评分
语音类数据处理 重点内容

语音数据特点：

  • 交互性强 - 适合语音问答、智能助手
  • 获取实时便捷 - 智能设备随时采集
  • 兼容性强 - 多数设备支持
  • 不直观 - 需算法解析
  • 失真风险 - 噪声、静音影响

预处理方法：

滤波、降噪、平滑、加重

语音段落切割规范：

  • 以说话人整句话为单位切割
  • 在语音波形图最低点设置边界
  • 不同说话人切割为不同语音段
  • 保留 2-3秒静音段

有效语音判定：

  • 规定之外的语言朗读
  • 纯噪声或静音
  • 声音极小无法听清
  • 语速过快或一字一顿
  • 存在切音、吞音、喷麦

业务流程五要素 核心考点
① 角色

第一个基本要素，有了角色才有分工、有协作，才能完成特定的业务目标

② 活动

指具体做的事，每个角色都有具体要做的事

③ 协作

不同人做不同事，通过协作完成工作

协作方式：并行 和 串行

④ 产出物

具体活动的结果，使不同活动在不同岗位之间的转手和交接成为可能

⑤ 规则

活动的内容、方式、责任等有明确的安排和界定
